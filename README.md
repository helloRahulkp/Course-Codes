# CourseCodes

## Overview

**CourseCodes** is a centralized repository that documents all the courses I have completed on Coursera.
The purpose of this repository is to systematically organize and showcase the **learning materials, certificates, notes, and implementation code** from each course.

## This repository acts as a **personal academic archive and technical portfolio**, allowing easy access to the knowledge and practical work completed during these courses.

## Repository Structure

Each course has its own folder inside the repository. Every folder may contain different types of learning materials collected during the course.

Typical contents include:

- **Certificates** – Official course completion certificates
- **Notes** – Personal study notes and summaries
- **Code** – Programming exercises and implementations
- **Jupyter Notebooks** – Practical experiments and demonstrations
- **Projects** – Course-related mini projects or assignments
- **Resources** – Additional references and materials used during learning

Example structure:

```
CourseCodes/
│
├── Course_Name_01/
│
├── Course_Name_02/
│
└── Course_Name_03/
```
